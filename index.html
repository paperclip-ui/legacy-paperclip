Paperclip
Build web application in no time.
Paperclip is a template language that comes with visual tooling to help you build UIs faster, bug-free, and more precicely.
Sign up for early access
Web development is slow & CSS tends to become unmanageable over time, especially as projects & teams become larger. Paperclip helps with that by providing you with tooling to keep track of visual changes, keep CSS tidy, and create your UIs in a visual editor.
Simple syntax
Paperclip uses familiar syntax, so there the learning curve is small. Use the visual editor to see your visual changes live.
<ol export component as="List">
<style>
padding-left: 1em;
font-family: Open Sans;
</style>
{children}
</ol>
<li export component as="ListItem">
<style>
margin-top: 6px;
</style>
{children}
</li>

No global CSS
Styles are scoped to the documents they're defined in, so you don't have to worry about them leaking out.
Live previews in VS Code
Paperclip comes with a live preview extension for VS Code, so you never have to leave your editor to see your HTML & CSS.
Incrementally adoptable
Paperclip compliments your existing codebase, so use it as you go.
See all of your UIs in one spot
No more digging around for UI elements. Open the birds-eye view to see all of your application UIs, and easily find what you're looking for.
Cross-browser testing made easy
Launch any browser directly within Paperclip and design against them in realtime.
Never miss a CSS Bug
Every single Paperclip UI is covered with visual regression tests - hardly any setup required. Just setup the Percy integration and you're good to go.
